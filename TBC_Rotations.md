# TBC Class Rotations Reference

This document contains single-target and AoE rotation information for LibSpellDB tagging.

## Classification System
- **CORE_ROTATION**: Abilities used constantly in single-target rotation (Row 1)
- **SITUATIONAL**: AoE abilities, big cooldowns, resource dumps (Row 2)

---

## Warrior

### Fury ST Rotation
1. Bloodthirst (on CD)
2. Whirlwind (on CD)
3. Rampage (maintain buff)
4. Execute (<20%)

### Fury Situational
- Heroic Strike (rage dump when >50 rage)
- Cleave (AoE rage dump)
- Sweeping Strikes (AoE)
- Death Wish (big CD)
- Recklessness (big CD)
- Bloodrage (resource)

### Arms ST Rotation
1. Rend (maintain)
2. Mortal Strike (on CD)
3. Overpower (when proc)
4. Whirlwind (on CD)
5. Slam (filler)
6. Execute (<20%)

### Arms Situational
- Heroic Strike (rage dump)
- Sweeping Strikes (AoE)

### Protection ST Rotation
1. Shield Block (maintain)
2. Revenge (on proc)
3. Shield Slam (on CD)
4. Devastate (filler/sunder stacks)
5. Thunder Clap (maintain debuff)

### Assassination ST Rotation
1. Slice and Dice (maintain)
2. Rupture (maintain)

4. Envenom (finisher)

### Subtlety ST Rotation

3. Rupture (maintain)
4. Eviscerate (finisher)

### Rogue Situational
